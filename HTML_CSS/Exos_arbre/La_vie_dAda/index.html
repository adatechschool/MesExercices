La vie d'Ada

Ada Lovelace, mathématicienne britannique du XIXe siècle, est considérée comme la première programmeuse de l'histoire pour avoir écrit le premier algorythme destiné à être éxécuté par une machine, la machine analytique de Charles Babbage, ce qui en fait une pionnière visionnaire de l'informatique moderne.

Le nom de l'école Ada Tech School s'inspire de cette mathématicienne.

En quelques points :

  • Pionnière de l'informatique : créa le premier algorythme pour ordinateur
  • Mathématicienne visionnaire du XIXe siècle
  • Collaboratrice de Babbage sur la machine analytique
  • Anticipait le potentiel des ordinateurs au-delà des simples calculs
